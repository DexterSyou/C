##CMD

```
/C ---------0----------1-----------2
  # cp source destination
    
  # cat 
    DES：显示文本文件中所有数据
	OPTIONS：
	     -n 加行号
		 -b 只有文本加行号
		 -T ^I字符组合替换指标符
/F ---------0----------1-----------2
  # file myfile
    DES：探测文件内部，并决定文件是什么类型  

/L ---------0----------1-----------2
  # ln -s filel file
  # ln file1 file
    DES：软连接，硬连接


/T ---------0----------1-----------2 
  # touch



/W ---------0----------1-----------2
  # wc
    DES：对数据文本进行计数，默认会输出3个值
                        文本的行数
                        文本的词数
                        文本的字节数
```
