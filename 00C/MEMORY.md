# 避免野指针
  避免野指针的方法，就是除了使用指针之外，都给指针初始化NULL

# 内存的三部分
  静态区  保存自动全局变量
          static变量
		  静态区的内容在整个程序的生命周期内都存在，由编译器在编译的时候分配
  栈      stack
          保存局部变量，只能在函数内作用
  堆      heap
          由malloc系列函数 或 new操作符分配的内存
		  由free 或 delete结束生命周期

          ////
          (void *) malloc(int size)
		  返回一个void 类型指针， 参数int类型数据（申请分配的内存大小）
		  malloc函数返回这块内存的首地址，申请的是连续的一块内存
		  if（NULL ！= p）//使用的时候必须校验
		  free（p） // 斩断p 和 那块内存的关联，p的值不变，
		               那块被释放的内存里里面保存的值没有改变，只是再无法使用
          free完指针之后，一定要给指针置NULL



# 函数的入口校验
