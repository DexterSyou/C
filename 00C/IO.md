# 文件访问
  FILE *fp 
  FILE *fopen(char *name, char *mode) 
    返回一个指向FILE的指针，如果发生错误，fopen将返回NULL
  int getc(FILE *fp)  
    返回fp指向的输入流中的下一个字符，如果达到文件尾或出错，该函数返回EOF
  int putc(int c, FILE *fp)
    该函数将字符c写入到fp指向的文件中，并返回写入的字符，如果发生错误返回EOF

  格式化的文件读写
  int fscanf(FILE *fp, char *format, ...)
  int fprintf(FILE *fp, char *format, ...)
  
  int fclose(FILE *fp)
  int ferror(FILE *fp)  如果fp中出现错误，则函数ferror返回一个非0值
  int feof(FILE *fp)    如果指定的文件到达文件结尾，它将返回一个非0值
  exit 为每个已打开的输出文件调用fclose函数，已将缓冲区中的所有输出写到相应的文件中

  启动c程序，操作系统打开3个文件，并将这3个文件的指针提供给该程序。
  <stdio.h>
   stdout    指向标准输出
   stderr    指向标准错误
   stdin     指向标准输入

# 行输入 和 行输出
  char *fgets(char *line, int maxline, FILE *fp)
  fgets 函数从fp指向的文件读取下一个输入行（包括换行符，并将它存放在字符数组line中，
  做多可读取maxline-1个字符，读取的行将以‘\0’结尾保存到数组中，通常返回line，遇到文件结尾
  或发生了错误，则返回NULL

  int fputs(char *line, FILE *fp)）
  输出函数fputs 将一个字符串（不包含换行符）写入到一个文件中，如果发生错误，该函数返回EOF
  否则返回一个非负值
