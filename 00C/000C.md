@计算机中数的表示
  Sign and Magnitude
  要用8bit表示正数和负数，最高位规定为符号位（Sign Bit 0正，1负），剩下的7位表示绝对值的大小
  8bit 取值范围是 -127 ～ 127

  1‘s Complement 负数用1的补码 
  2‘s Complement 正数不变，负数先取反再加1

@数据类型
 C 优先考虑效率，而移植性在其次
 C 语言与 平台 ，编译器是密不可分的 
 一般来说char型表示8位的整数，为了可移植性就必须写明是signed 还是 unsigned

C标准中没有做明确规定的地方会用（未明确定义的）
  Implementation
     如C 标准中没有明确规定char是有符号的还是无符号的，但是要求编译器必须对此明确规定
  Unspecified
     C 标准没有明确规定按哪种方式处理，编译器可以自己决定
  Undefined
     完全不确定的，C标准没有规定，编译器很可能也没有规定，甚至没有出错处理
  
 除了char型以外的这些类型如果不明确指定signed或unsigned，都是signed C的标准明确规定的
 除了char型在C中明确规定占一个字节之外，其他整形占几个字节都是implementation Defined
