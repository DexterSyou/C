# 链接器
  把有编译器 或 汇编器 生成的若干个目标模块，整合成一个被称为 可执行文件 的实体
  链接器通常把 目标模块 看成由一组外部对象组成的。程序中每个函数，每个外部变量，如果没有
  声明static，就是一个外部对象

  int a； 定义外部变量a
  extern int a 声明变量a  是对一个外部变量a的显示引用，即使出现在一个函数内，也具有同样的含义

  如果有两个定义
    int a 要么错误，要么a为同一个实例

  ANSI C 允许在声明时指定函数的参数类型
