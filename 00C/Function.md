函数的定义形式：

 returnType FunName(ParameterList){
     statements;	
     return 表达式;
 }

 returnType: 默认返回int类型
 
# 外部变量
 1.external 外部变量（定义在函数之外变量）
            函数本身（函数中不允许定义其他函数）
   外部变量 与 函数 具有下列性质：
      1） 外部链接： 通过同一个名字对外部变量的所有引用
      2） 外部变量可以在全局范围内访问
	  3）

 2.internal 定义在函数内部的函数参数及变量

# 作用域



 局部变量         ： 函数区域
 外部变量 或 函数 ： 从声明它的地方开始，到其所在的（待编译）文件的末尾结尾
 
 外部变量的定义与变量的使用不在同一个源文件，则必须在相应的变量声明中使用extern
 变量声明： 变量的属性（主要是变量的类型）
            告诉编译器，这个变量已经匹配到了一块内存上
			告诉编译器，这个变量或对象我先预定了，别的地方不可以使用了。
 变量定义： 除变量属性外，还引起存储器的分配
            编译器创建一个对象，为这个对象分配一块内存空间并给他取上一个名字

 在一个源程序的所有源文件中，一个外部变量只能在某个文件中定义一次，
 而其他文件可以通过extern声明来访问它。初始化只能在定义的时候

# 静态变量
 用static 声明限定外部变量 与 函数，将其后声明的对象的作用域限定为被编译源文件的剩余部分。
 static的内部变量，只能在某个特定的函数中使用但一直占据存储空间的变量
