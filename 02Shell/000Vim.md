## Vim 

```
#******************************************************
# 基础
#******************************************************

@光标移动
单位级 
      h左 l右
      k上
      j下
单词级
     w W 向右移动到下一个单词开头
	 e E 向右移动到单词结尾
	 b B 向左移动到单词开头
	 小写以分词作为单词界限，大写以空格为界限
块级
     gg 文档第一行
	 G  文档最后一行
	 0  行首（第一列）
	 $  行尾 
	 ^  到第一个非空白字符
     H
	 M
	 L
     Ctrl-d 向下移动半页
	 Ctrl-u 向上移动半页
	 Ctrl-f 向下移动一页
	 Ctrl-b 向上移动一页
     :<N> or <N>gg 跳转到N行
     :+<N> or <N>j
	 :-<N> or <N>k

@内容查找，打开文件
 在Vim 中打开文件
 ：e <filename>
   e #  切换到上一个文件
   r #  把上一个文件内容复制到当前文件
   e！# 放弃当前文件编辑
 ：Ex 在Vim中打开目录树，光标选中后回车打开对应文件 （- 进入上级目录）

 文档内查找
  * 向后查找光标当前所在单词
  # 向前
  /<search> 向后查找指定字符串
  ？<search>
  n 方向性 下一个
  N 方向性 上一个

 行内查找
  f（X）向行尾查找并定位到字符X
  t（X）向行尾查找并定位到字符X之后
  F（X）向行首查找并定位到字符X
  F（X）向行首查找并定位到字符X之后
  ； 当前方向查找下一个字符
   ，当前反方向查找下一个字符

 %可以对以下匹配符号进行匹配查找，找到匹配的另外一个符号
  （  ）[ ] { }
    
 ：bn
 ：bp
 ：b<N>

@ 文档的修改 与 保存
 插入
  i
  a 当前字符后插入
  I 行首插入
  A 行尾插入
  o 在下一行插入
  O 在上一行插入
 
 ESC 
   Ctrl+C
 
 删除
   s 删除当前字符，并进入插入模式
   S 
   x 删除当前字符
   X 删除前一个字符
   dd
   d<X> 删除指定内容
   cc 删除当前行并保存到Vim剪贴板上，并进入Insert模式
   c<X> 删除字指定内容.....

 复制 粘贴
  yy
  y<X> 复制指定内容
  p 当前位置后
  P 当前位置前

 合并
   J 将当前行与下一行合并
  
 替换
  r<X>
  gu<X> 
  gU<X>
  ：%/<search>/<replace>/

 撤销
  u
  Ctrl+r

 保存
  w
  wa
  wq
  q
  saveas
  w <new filename>


 缩进
   >>
   <<
   <g












```
